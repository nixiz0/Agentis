Installation Guide of Agentis

Open Ubuntu Shell and run this commands :

  • curl https://ollama.ai/install.sh | sh
  • ollama serve

Attention
If you have this message :
Error: listen tcp 127.0.0.1:11434: bind: address already in use

Then execute this command :
sudo systemctl stop ollama
(Sudo will ask your user password
that you set when installing WSL for Ubuntu)

Download & Use Local Models

You click on Start button
On the input you have to enter the model you want

Attention
You have to download previously the model in your machine

How ?
Go on this website : ollama.ai/library.
Find models you want, and go in your Ubuntu shell and run :
ollama pull 'name model'

You can return on Agentis Menu and click on start
Then put on the input the name of one model that you have downloaded

If you want to see all the models that you have downloaded
Run this command on your Ubuntu shell :
ollama list

Download History
The HTML History button allows you to download your conversation
with your model and stores it in your "download" folder on your computer

The CSV History button allows you to download your conversation
with your model and stores it in your "download" folder on your computer

Also you have on the app interface in Python the
button "Open History"
which allows direct access to the download file where
you will find your histories in html and or csv

Essential Commands Ollama

ollama serve
Start Ollama serve API.

ollama list
Show all your downloaded models.

ollama pull 'name model'
Download a specific model.

ollama run 'name of the model'
Run a specific model on the terminal.

ollama rm 'name model'
Delete a model.

If you want to see more commands
just write on your Ubuntu shell
Ollama
